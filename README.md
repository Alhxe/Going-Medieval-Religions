# Going Medieval Religions

Religion mod framework for Going Medieval — adds new religions with custom structures, items and rituals. Currently includes Christianity.

## Stack

- Unity Editor **2022.3.46f1** (with Windows Build Support)
- Going Medieval **0.23.3**
- Based on [Foxy Voxel modding template](https://github.com/FoxyVoxel/going-medieval-modding)

## Build

1. Open this project in Unity 2022.3.46f1
2. `Going Medieval` menu → `AddressableBuilder` → select mod → `Create Addressables` → `Build`
3. Copy contents of mod's `Exported/` folder to `Documents/Foxy Voxel/Going Medieval/Mods/<ModName>/Data/AddressableAssets/`
4. Launch game → enable mod in menu
